Description: Implement an application to solve the N-Queens problem with n set to 8. Use two algorithms in implementation: Steepest-Ascent Hill Climbing and CSP Min-Conflict. 

### Run Program Steps
1. open IDE you would like to use, these directions use <i>Eclipse</i>
2. create a new project
<br><i>note: finish, you don't need a module</i><br>
3. create a new package named `nQueens`
4. right click package and select `import`
5. General -> File System -> Browse
<br>`> import .java files`
6. run the main class
7. select option given from menu
